Folio Privacy Policy

Last updated: June 25, 2026

Overview

Folio is a browser extension that replaces the new tab page with a customizable interface including widgets, AI search shortcuts, background customization, and optional integrations.

Folio is designed to operate locally inside your browser.

Data Collection

Folio does not collect, transmit, sell, or store personal data on external servers.

All user data remains on your device using Chromium's local storage APIs. No data is shared except to certain third parties for integration purposes.

Data Stored Locally

Folio may store the following data locally in your browser:
  • Custom CSS for the new tab page
  • Background image settings and uploads
  • Widget settings and layout positions
  • AI provider selection (ChatGPT, Claude, Grok, etc.)
  • Feature toggles (history, search bar, redirect, etc.)
  • Integration settings (Todoist, Weather)
This data never leaves your device.

Permissions

Storage
Used to save settings locally on your device.
History
Used only to display your recent browsing history in the new tab page.
Tabs
Used to open pages or AI search results when triggered by you.
Active Tab
Used only when interacting with the current tab via user-initiated actions.
Context Menus
Used to enable right-click actions such as “Ask AI about selected text”.

Integrations

Folio includes optional integrations that only run when explicitly enabled by the user.
Todoist Integration
When enabled, Folio loads a Todoist ICS feed provided by the user.
  • The ICS URL is stored locally in your browser.
  • No Todoist account data is stored by Folio.
  • Requests are made directly from your browser to show the tasks from the Todoist ICS feed.
  • No personal data is included in any requests to todoist, except the ICS link provided by the user.
  • Any data handling beyond Folio is governed by Todoist’s privacy policy
Weather Integration
When enabled, Folio fetches weather data from Open-Meteo.
  • Uses browser geolocation if permitted by the user
  • Requests are only sent to Open-Meteo if the user has enabled the weather integration.
  • Requests go directly to Open-Meteo API
  • Your anonymous IP address is used by Open-Meteo to return accurate weather information.
  • Any data handling is governed by Open-Meteo’s privacy policy

AI Services

When you use AI search features, your query is sent directly from your browser to the selected AI provider. Supported providers may include ChatGPT, Claude, Grok, Perplexity, Mistral, and Microsoft Copilot.
  • Folio does not store or intercept prompts
  • Requests are not processed by Folio servers (none exist)
  • Each provider’s privacy policy applies

Third-Party Services

Folio may send requests to external services only when you explicitly use features that require them (AI providers, weather API, Todoist feed). No background tracking or silent data transmission occurs.

Contact

For questions about this privacy policy, refer to the extension listing or project page where this extension is distributed.
