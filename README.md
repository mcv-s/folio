# Folio
Folio - Ultimate New Tab



Folio is a Chromium extension which adds several features to your web browser. It supports:

An AI search bar
Custom backgrounds
Random backgrounds
Widgets/Integrations

. . . and a lot of simplicity, because most new tabs just have too much.



To download, you can go to https://github.com/mcv-s/Folio/archive/refs/heads/main.zip
